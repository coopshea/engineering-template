# Competitive Reference

Public-domain reference material for prior art and predicate device research. **All sources must be freely available** (USPTO, Google Patents, FDA 510(k) database, manufacturer publications). No confidential or licensed material.

## Structure

| Folder | Contents |
|--------|----------|
| `devices/` | Competitor product specs and dimensions, one file per product |
| `patents/` | Published patent references — bulk index in `patents/README.md`, detailed notes only for key patents |
| `assets/` | Images extracted from patents, device photos |

## See Also

- `AGENTS.md` (root) — full structure conventions and bulk patent entry table format
- `../sources.yaml` — citation file; competitive references should also have entries here for traceability
